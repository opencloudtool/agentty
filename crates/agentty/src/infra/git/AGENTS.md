# Git Infrastructure

## Overview

Workflow-focused git operations for merge, rebase, repository state, sync, and worktree management.

## Directory Index

- [merge.rs](merge.rs) - Merge-specific git command orchestration helpers.
- [rebase.rs](rebase.rs) - Rebase workflow, resume handling, and related command orchestration.
- [repo.rs](repo.rs) - Repository discovery, inspection, and shared git utilities.
- [sync.rs](sync.rs) - Pull/rebase/push synchronization and main branch sync operations.
- [worktree.rs](worktree.rs) - Git worktree creation, cleanup, and branch lifecycle helpers.
- [AGENTS.md](AGENTS.md) - Local agent instructions and directory index.
- [CLAUDE.md](CLAUDE.md) - Symlink to AGENTS.md.
- [GEMINI.md](GEMINI.md) - Symlink to AGENTS.md.
