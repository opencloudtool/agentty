# Git Infrastructure

Git operation helpers used by the infrastructure layer.

## Directory Index

- [merge.rs](merge.rs) - Merge-specific git command helpers.
- [rebase.rs](rebase.rs) - Rebase workflow and resume helpers.
- [repo.rs](repo.rs) - Repository discovery and generic git utilities.
- [sync.rs](sync.rs) - Pull/rebase/push synchronization helpers.
- [worktree.rs](worktree.rs) - Git worktree and branch lifecycle helpers.
- [AGENTS.md](AGENTS.md) - Local instructions and directory index.
- [CLAUDE.md](CLAUDE.md) - Symlink to `AGENTS.md`.
- [GEMINI.md](GEMINI.md) - Symlink to `AGENTS.md`.
