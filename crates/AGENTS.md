# Crates Directory

Contains the workspace member crates.

## Workspace Crates

- `crates/ag-forge/` holds forge review-request types and provider integrations shared across the workspace.
- `crates/ag-xtask/` holds workspace maintenance utilities, including migration checks and generated workspace-map output.
- `crates/agentty/` is the main TUI application crate with `app`, `domain`, `infra`, `runtime`, and `ui` layers.
- `crates/testty/` provides the Rust-native TUI end-to-end testing framework.
