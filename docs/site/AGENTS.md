- [`content/`](content/) - Zola content files.
- [`static/`](static/) - Static assets served directly.
- [`templates/`](templates/) - Zola HTML templates.
- [`config.toml`](config.toml) - Site configuration.
- [`sass/`](sass/) - Sass source for site styling and compiled assets.

- [`GEMINI.md`](GEMINI.md) - Symlink to AGENTS.md.
