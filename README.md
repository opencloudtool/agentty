# Agentty

A terminal UI tool for managing agents, built with Rust and [Ratatui](https://ratatui.rs).

## Prerequisites

- Rust nightly toolchain (configured via `rust-toolchain.toml`)

## Quickstart

```sh
git clone <repo-url>
cd agentty
cargo run
```

## Development

```sh
cargo test
cargo clippy -- -D warnings
cargo fmt --all -- --check
```
